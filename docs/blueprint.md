# **App Name**: GradeAid

## Core Features:

- Exam Input: Accept student exam submissions, either through direct upload or integration with a learning management system. Accepts student's exam.
- Evaluation Criteria Input: Allow instructors to input evaluation criteria, including points per question, grading rubrics, and specific keywords or concepts to look for.
- AI-Powered Grading: Use a generative AI tool to automatically grade exams based on the evaluation criteria. It uses reasoning to decide whether the answers respond to the grading rubrics. 
- Personalized Feedback Generation: Generate personalized feedback for each student, explaining the rationale behind the grade received and areas for improvement using generative AI.
- Grade Display: Display each studentâ€™s grade along with the detailed feedback on a dedicated webpage.
- Downloadable PDF Report: Generate a downloadable PDF report for instructors containing all grades, detailed feedback, and submission details for each student.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) for a sense of knowledge and authority.
- Background color: Light Greyish-Blue (#E8EAF6), almost white, for a clean, readable interface.
- Accent color: A vibrant Yellow-Orange (#FFAB40) to highlight important feedback and call-to-action buttons.
- Body and headline font: 'Inter' (sans-serif) for a clean and modern look that ensures readability.
- Use minimalist, professional icons for navigation and to represent different sections or features, focusing on clarity.
- Design a clean, structured layout with clear visual hierarchy, making it easy for instructors and students to find information quickly.
- Incorporate subtle animations, like fade-ins and transitions, to enhance user experience without distracting from the primary task.
